Inicio

Para uma breve compreensão da solução apresentada para o projeto "Room Booker" apresentamos as distintas funcionalidades da mesma.
Realizado por: André Silveira, Daniel Borges e Rafael Figueiredo

Website

Para facilitar o acesso aos dados do projeto foi criada uma API, permitindo a conexão entre base de dados e aplicações de forma segura e eficiente.
A API utiliza endpoints, que são simplesmente URLs que especificam um determinado recurso ou serviço.
Utilizamos esses endpoints para estabelecer comunicação a esse serviço específico e executar a sua função.
Funções como, obter, criar e atualizar dados de utilizadores, salas, centros e reservas.
A autenticação do utilizador é também feita através da API, obtendo um token de sessão com a informação do utilizador e as suas permissões no que diz respeito ao acesso aos dados.
A aplicação web foi desenvolvida com HTML, CSS e JavaScript, utilizando as suas ferramentas e tecnologias como o Bootstrap, Node.js, React e Heroku, no ambiente de desenvolvimento Visual Studio Code.
O Bootstrap ajuda a facilitar o processo de design do WebSite, tendo já classes predefinidas prontas a usar, com as suas configurações de CSS. Facilita o uso do CSS.
O React é uma framework que torna fácil a criação de UIs interativas. Ajuda a atualizar com facilidade e dar render aos componentes certos quando a informação muda.
Para postar o WebSite online e para toda a gente ver, utilizámos o Heroku que facilita esse processo e um guia com várias maneiras de o poder efetuar.

App Mobile

A aplicação principal do projeto foi feita em java no android studio e está disponível a todos os tipos de utilizador, desde o admin até ao utilizador normal.
O objetivo desta aplicação é permitir ao utilizador ver as salas disponíveis para cada centro ao qual está associado.
Permitindo então reservar essas salas consoante a sua disponibilidade.
Essa disponibilidade é verificada percorrendo todas as restantes reservas e comparando se existe interposição de horários tendo em conta o tempo de limpeza.
O utilizador tem também acesso às suas reservas correntes onde pode alterar a mesma de acordo também com a disponibilidade da sala e eliminar a reserva.
Tem ainda acesso ao histórico de reservas, onde não pode efetuar alterações, mas tem acesso a essa informação.
Pode também alterar o seu perfil e por último tem um scanner de QRCodes, que será utilizado juntamente com a app criada para o tablet, e permite ao fazer o scan da sala ir diretamente para as reservas dessa sala.

App Tablet

A segunda aplicação tem como função ser instalada nos tablets expostos à frente de cada sala.
O seu objetivo é mostrar as reservas da sala correspondente entre duas datas que podem ser especificadas, caso contrário irá mostrar as reservas do dia atual.
Irá também mostrar os detalhes da sala e ao fazer login com uma conta de admin ou gestor é possível alterar a informação da sala.
Para ser selecionada a sala que será demonstrada na aplicação é necessário fazer login com uma conta de admin, gestor ou de limpezas.
A aplicação gera também um qr code com a informação da sala que irá levar o utilizador da aplicação móvel para as reservas da sala correspondente.
